## Problem

A Slovak SaaS startup had a React MVP, but the marketing site was "later, by the founder, in WordPress." Lots of duplicated CTAs, mixed fonts, no analytics, weak SEO. For Series A they needed professional-looking launch pages.

## Solution

Migration to Astro for the marketing side (landing, pricing, blog, docs), keeping the React MVP at app.horalka.io. Stripe checkout in Astro for the top-up pricing tier (no app login needed). Plausible Analytics, per-post OG image generation.

## Result

- Lighthouse 100/100/100/95 (perf/a11y/best/SEO)
- Q4 2025 sales +60% vs Q3 (mix of factors, but tracking showed direct organic +180%)
- Founders edit the blog via GitHub web UI from markdown — no CMS costs
