## Problém

Slovenský SaaS startup mal MVP v Reacte, ale marketingový web bol "later, by the founder, in WordPress." Veľa duplikovaných CTA, zmiešané fonty, žiadny analytics, slabý SEO. Pre Series A potrebovali profesionálne pôsobiace launching stránky.

## Riešenie

Migrácia na Astro pre marketingovú časť (landing, pricing, blog, docs), zachovanie React MVP pre app.horalka.io. Stripe checkout v Astro pre top-up pricing tier (žiadny app login potrebný). Plausible Analytics, OG image generation per blog post.

## Výsledok

- Lighthouse 100/100/100/95 (perf/a11y/best/SEO)
- Sales Q4 2025 +60 % oproti Q3 (mix faktorov, ale tracking ukázal direct organic +180 %)
- Foundery editujú blog z markdownu cez GitHub web UI — žiadne CMS náklady
