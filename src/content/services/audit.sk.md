Pozriem tvoj existujúci web cez 4 lupy: performance, SEO, accessibility, bezpečnosť. Dostaneš písomný report s prioritizovaným zoznamom — najprv kritické (web padá / nedáva sa použiť), potom dôležité (znižuje konverziu / SEO score), nakoniec nice-to-have (drobnosti).

Performance: Lighthouse na 5 reprezentatívnych stránok, WebPageTest na 3G/4G profiloch, Core Web Vitals zo Search Console. Identifikujem render-blocking resources, neoptimalizované obrázky, third-party script bloat, layout shift culprits.

SEO: hreflang ak máš multilingual, schema.org markup, internal linking graph, canonical tags, sitemap completeness, meta description coverage. Často nájdem 10-20 stránok bez meta description — drobnosť, ale -10 % CTR z Googlu.

Accessibility: axe automated audit + manuálne keyboard testing (5 najdôležitejších flows), color contrast, ARIA roles, focus order. Zákony EÚ tlačia accessibility do compliance — lepšie sa pripraviť teraz ako platiť pokuty neskôr.

Reporty doručujem do 5-7 pracovných dní. Po dodaní 30-min call kde prejdeme čo a v akom poradí riešiť.
