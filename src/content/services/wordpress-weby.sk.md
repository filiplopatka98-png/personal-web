WordPress má povesť pomalého CMS-ka. Zaslúženú? Sčasti — ale po piatich rokoch ladenia WP webov môžem povedať, že vie byť rýchly, len musíš vedieť, čo robíš.

Stavám custom themes od nuly. Žiadny Elementor, Divi ani WPBakery. Tieto page-buildery generujú nadbytočný HTML/CSS/JS, robia web ťažkým a ťažko spravovateľným po roku-dvoch. Namiesto toho používam Gutenberg block library prispôsobenú konkrétnemu klientskemu workflow — predpripravené bloky pre produktové kategórie, kampaňové stránky, blog rozvrhnutia.

Web po odovzdaní vie klient sám editovať. Žiadne „treba zavolať vývojára" pre zmenu textu na hlavnej stránke. Dokumentáciu pripravujem priamo počas projektu, hand-off call rieši zvyšné otázky.

WooCommerce ak treba e-shop — viem nastaviť payment gateway-y (Stripe, PayPal, GoPay), VAT settings pre EÚ, multilingual setup cez Polylang alebo WPML. Performance optimalizujem cez WP Rocket + ShortPixel + custom critical CSS. Lighthouse 90+ je východisko, nie bonus.
