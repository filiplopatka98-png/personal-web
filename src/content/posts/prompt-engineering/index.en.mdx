Over the past year I've spent more time with AI-assisted code than without. What I've learned:

## Write specs, not commands

Bad prompt: "Make me a form in Astro."

Good prompt: "Astro component FormContact, props: action (string), method (POST/GET). Fields: name (required, min 2), email (validated), message (textarea, required min 10). Submit via fetch, success state inline, error state under field. Tailwind, our tokens in var(--color-accent). No external state lib."

The second works 9 times out of 10. The first 1 time out of 10 and you'll rewrite it anyway.

## Drag the model into your style

Best trick: at the start, show the AI 2-3 existing components from the project. "Here's BaseLayout, here's Header, here's MobileMenu. Match this pattern in the new component." Models (Claude, GPT-4) maintain consistency much better with context than without.

## What I never AI

- Auth flow (security-critical, requires depth of understanding)
- Database migrations (one bad query = downtime)
- Build config (Astro/Vite/PostCSS — models hallucinate APIs)

## Cost reality check

Anthropic API + Cursor + ChatGPT: ~$60/month for solid productivity. Pays back at the first 2-hour saving in a week. Worth it.
